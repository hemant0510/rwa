# External Documents Overview

This directory contains various documents used by the Eden RWA project. Most files are in Microsoft Word (`.docx`) or Excel (`.xlsx`) formats and represent forms, responses, and official letters.

## Contents

- `2017-05-0000335.docx` – likely an official letter or reference from May 2017.
- `DHBVN (Responses).xlsx` – an Excel sheet capturing responses from DHBVN (electricity provider).
- `DHBVN_13thJune2025.docx` – correspondence with DHBVN dated June 2025.
- `Eden Estate RWA form (Responses).xlsx` – membership or survey responses for Eden Estate RWA.
- `Eden RWA.xlsx` – general RWA data in spreadsheet form.
- `EERWA Collection 2025-26.xlsx` – collection records for 2025–26 financial year.
- `EERWA General Body.xlsx` – general body meeting data.
- `EEWA FORM.docx` – a form used by the RWA (likely for residents).
- `EEWA_Membership_form_for_Owners.docx` – membership form targeted at property owners.
- `Letter for Safety.docx` – a safety-related letter.
- `RWA Collection 2024-25.xlsx` – collection records for 2024–25 year.

> **Note:** The contents of these files are not directly accessible via plain text. To view or convert them, open them in Microsoft Office or a compatible viewer.

If specific data from any document is needed for code or integration, please open the file manually and extract the required information.
